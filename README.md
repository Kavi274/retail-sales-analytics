# Retail Sales Analytics Platform

## 📊 Project Overview

This project analyzes retail sales data using SQL and Power BI to generate business insights and visualize performance trends.

## 🛠 Tools & Technologies

* SQL (SQLite)
* Power BI
* CSV Dataset (Superstore)

## 🔍 Key Analysis Performed

* Total Revenue, Profit, and Quantity analysis
* Category and Sub-category performance
* Regional sales analysis
* Top customers and products identification
* Monthly and yearly sales trends

## 📈 Dashboard Features

* KPI Cards (Revenue, Profit, Quantity)
* Monthly Sales Trend
* Sales by Category and Region
* Top Customers and Products
* Interactive filters (Region, Category, Date)

## 💡 Key Insights

* Identified top-performing categories and regions
* Observed seasonal sales trends
* Found high-value customers contributing to revenue
* Highlighted profitable and loss-making segments

## 🚀 Outcome

Developed an end-to-end retail analytics solution combining SQL and data visualization for business decision-making.
